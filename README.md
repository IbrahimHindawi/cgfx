# cgfx
## C graphics programming template with SDL2 OpenGL  

Template project for graphics programming:  
Use scripts in `scripts` folder to build compile and run.  

Dependencies:  
* vcpkg  
* SDL2  
* GLAD  
* CGLM  
* STB image  
* vcpkg  
* MSBuild  
* Visual Studio 2022
